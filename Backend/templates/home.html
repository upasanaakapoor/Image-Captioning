    • Image Caption Generator

    Image Caption generator is a challenging Artificial Intelligence based task where a textual description for a given image is provided by the model. This caption generator involves methods from computer vision to natural language processing concepts to recognize the contents of an image and describe them in a natural language like English in the right order. The model is trained so as to maximize the likelihood of the target description sentence given the training image.
